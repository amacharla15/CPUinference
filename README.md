# Project 1 - CPU Inference Server + Instrumentation

## Goal
Build a CPU inference server that accepts prompt-based requests, streams output, and records latency and memory metrics.

## Planned phases
- Phase 0: Setup
- Phase 1: Serving boundary
- Phase 2: HTTP server
- Phase 3: Streaming output
- Phase 4: Timing instrumentation
- Phase 5: RAM measurement
- Phase 6: Benchmark harness
- Phase 7: Prompt-length experiments
- Phase 8: Result storage
- Phase 9: README writeup
